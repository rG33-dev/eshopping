## Setup & Installation

1. **Clone the repository**
    ```bash
    git clone https://github.com/rG33-dev/eshopping.git
    cd eshopping
    ```
2. **Install dependencies**
    ```bash
    # Frontend
    cd frontend
    npm install

    # Backend
    cd ../backend
    npm install
    ```
3. **Configure environment variables**
    - Copy the example `.env.example` to `.env` and update the values as needed in both frontend and backend directories.

4. **Run the project**
    ```bash
    # In frontend/
    npm start

    # In backend/
    npm start
    ```

5. **Access the app**
    - Visit [http://localhost:3000](http://localhost:3000) in your browser for the frontend.
    - Backend runs on [http://localhost:5000](http://localhost:5000) (or as configured).
